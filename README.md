# Yalv
### Install Package
 * yarn or npm install
 
### Run Docker
  * yarn dockerBuild
  * yarn dockerRun

### Run App
 * yarn start (expo start)
 * yarn ios (expo start ios)
 * yarn android (expo start android)

### Run WebPack
 ##### Mode Developemnt: 
  * yarn buildDev
 ##### Start Server
  * yarn server
 ##### Eslint Check
  * yarn lint
  * yarn lint --fix (auto fix)
