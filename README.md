# Yalv
### Install Package
 * yarn or npm install
### Running App
 * yarn start (expo start)
 * yarn ios (expo start ios)
 * yarn android (expo start android)

### Running WebPack
##### Mode Developemnt: 
 * yarn buildDev
##### Running Server
 * yarn server
##### Eslint Check
 * yarn lint
 * yarn lint --fix (auto fix)
